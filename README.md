# react-tree

> NOT READY

> React component that represents a tree from a given array consistent with d3js tree examples.

## Building

For developing:

```sh
$ npm run dev
```

For prodution:

```sh
$ npm run build
```

### Examples

Go to the `/examples` folder and then:

```sh
$ webpack
$ http-server
```

Now you're ready to rock! :neckbeard:
